# 📊 Data Visualization Project

## 📌 Overview

This project demonstrates how to transform raw data into meaningful visual insights using Python. The dataset is cleaned, analyzed, and visualized with different types of charts to better understand book price distribution and trends.

---

## 🎯 Objectives

* Load and explore a CSV dataset.
* Clean and preprocess the data.
* Perform basic statistical analysis.
* Create informative visualizations using Matplotlib and Seaborn.
* Save all generated charts for future use.

---

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Data-Visualization-Project.git
```

Move into the project folder:

```bash
cd Data-Visualization-Project
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python visualization.py
```
